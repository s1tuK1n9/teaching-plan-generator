# 教案自动生成与模板填充 Skill

## 谁可以用

不需要懂技术。只要你有一份学校发的 Word 教案模板，再加几份以前写好的教案做参考，AI 就能帮你自动生成新的教案。

**关键原则：AI 完全根据你提供的模板和参考教案来动态适应，不会预设任何固定格式。**

## 触发条件

当用户提到以下关键词时激活：
- "生成教案"、"写教案"、"教案模板填充"、"教案制作"
- 提供"周次 + 主题 + 知识点"时
- 提供 PPT 文件要求"根据PPT制作教案"时

## 核心原则：模板自适应

**不预设任何模板结构。** 每次运行，先读取用户提供的 Word 模板，通过标签文本匹配来理解模板结构，动态建立映射表，然后填充。无论模板是几行几列、什么布局、哪个学校用，只要包含教案标准标签（"课程名称""教学目标""学情分析"等），Skill 都能自动适配。

同样，**不预设教学风格。** 每次运行，先扫描用户提供的参考教案，从中学习该教师的写作习惯（动词风格、篇幅长短、反思格式等），然后在生成新教案时模仿这种风格。

## 执行流程

### 第〇步：检测输入模式

- **提供了 .pptx** → PPT 模式：提取幻灯片内容，分析章节，推断知识点
- **仅文字描述** → 文字模式：直接进入下一步

### 第一步：动态分析模板结构

用 PowerShell 打开用户提供的模板 `.docx`，逐表格、逐行、逐列扫描，通过**标签关键词匹配**建立字段→单元格映射表。

**匹配规则（按优先级）**：
1. 精确匹配：单元格文本等于标签（如"课程名称"）
2. 包含匹配：单元格文本包含标签（如"课程名称："包含"课程名称"）
3. 关键词匹配：单元格文本包含关键词（如出现"任课教师"→匹配 teacher 字段）

**标签→字段对照表**：

| 模板中的标签（支持模糊匹配） | 映射字段 |
|---|---|
| 课程名称 | `course_name` |
| 课程代码、课程编号 | `course_code` |
| 授课班级、班级、教学班级 | `class_name` |
| 所属专业、专业 | `major` |
| 开课学院、学院、教学部门 | `college` |
| 主讲教师、任课教师、教师 | `teacher` |
| 项目名称、课题名称、教学项目 | `project_name` |
| 授课类型、课程类型、课型 | `course_type` |
| 周次、教学周 | `week_info` |
| 学时、课时 | `hours` |
| 知识目标 | `knowledge_objective` |
| 能力目标、技能目标 | `ability_objective` |
| 素质目标、素养目标 | `quality_objective` |
| 学情分析 | `student_analysis` |
| 教学重点 | `teaching_focus` |
| 教学难点 | `teaching_difficulty` |
| 教学方法、教学策略 | `teaching_methods` |
| 教材资源、教学资源、教材 | `resources` |
| 课前（行/区域标题） | 课前区域起点 |
| 课中（行/区域标题） | 课中区域起点 |
| 课后（行/区域标题） | 课后区域起点 |
| 课程导入、导入 | 课中环节1 |
| 知识点一、知识一、新知讲解 | 课中环节2 |
| 知识点二、知识二、技能训练 | 课中环节3 |
| 互动、练习、实践 | 课中环节4 |
| 小结、总结、布置作业 | 课中环节5 |
| 教学反思 | `reflection_area` |
| 教学效果 | `teaching_effect` |
| 诊断改进、诊断 | `diagnosis` |
| 改进 | `improvement` |

**映射建立逻辑**：
- 找到标签单元格后，其右侧或下方相邻的单元格即为值单元格
- 如果是"教学目标"区域，其下方3行分别对应知识、能力、素质（按标签二次匹配）
- 如果是"课前/课中/课后"区域，区域内识别"教学内容""学生活动""教师活动""设计意图"四个子列

### 第二步：读取模板预填值 + 合并用户输入

扫描模板中已填写的值，识别有效值（非空、非占位符如 `Xx` `xxxx` `??` `待填`），按优先级合并：

```
用户本次明确指定 > 模板已预填的有效值 > 参考教案中最常见值
```

### 第三步：学习参考教案风格

扫描项目目录下所有 `.docx` 文件（排除模板本身），提取文本，分析：
- 教学目标的动词分层（掌握/理解/了解、能够/熟练）
- 学情分析的段落结构
- 课中五环节的内容深度和篇幅
- 教学反思的效果数据+诊断+改进格式

优先选择与当前主题相近的参考教案。

### 第四步：生成完整教案内容

基于 topic、kp1、kp2，参照参考教案风格和第二步的动态映射，生成全部模块：
- 教学目标（知识4条 + 能力4条 + 素质2-3条）
- 学情分析、教学重难点、教学方法、教材资源
- 课前/课中五环节×四维度/课后教学设计
- 教学反思（效果含数据 + 诊断 + 改进，带"诊断："和"改进："前缀）

### 第五步：复制模板并填充（直接操作 Word COM）

AI 直接通过 PowerShell 操作 Word COM 完成填充，无需依赖外部脚本。

**5.1 复制模板**
```powershell
# 找到模板文件（体积最小的 .docx 通常是模板）
$t = Get-ChildItem "*.docx" -File | Sort-Object Length | Select-Object -First 1
Copy-Item -LiteralPath $t.FullName -Destination "第X周_教案.docx" -Force
```

**5.2 打开 Word 并写入内容**
```powershell
$word = New-Object -ComObject Word.Application
$word.Visible = $false
$doc = $word.Documents.Open((Resolve-Path "第X周_教案.docx").Path)

# 逐表格遍历，使用第一步建立的标签→位置映射表
# 每个找到的标签单元格，向其对应的值单元格写入内容
# 例如：
# $doc.Tables.Item(1).Cell(2, 3).Range.Text = "计算机网络"

$doc.Save()
$doc.Close()
$word.Quit()
[System.Runtime.Interopservices.Marshal]::ReleaseComObject($word) | Out-Null
```

AI 在执行填充时，严格依据第一步扫描建立的字段→单元格（Table/Row/Col）映射表来定位写入位置，确保每个字段填入正确的位置。

## 重要约束
- 不预设模板结构，全部通过标签匹配动态适配
- 不预设教学风格，通过分析用户提供的参考教案来模仿
- 内容具体可执行，教学环节时间不超过 90 分钟
- 严格保持模板表格、字体、页眉页脚、页面布局不变
- 思政元素自然融入
- 输出命名：`第X周_教案.docx`
- 教学反思的"诊断："和"改进："需带前缀
